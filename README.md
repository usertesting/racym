Rails Application Configuration for Yield Main (RACYM)

racym is used as a shortcut to rails configuration.

## Installation

Add this line to your application's Gemfile:

    gem 'racym'

And then execute:

    $ bundle

Or install it yourself as:

    $ gem install racym
